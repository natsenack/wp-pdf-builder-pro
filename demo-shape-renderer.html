Démo ShapeRenderer - Phase 3.3.3

Cette démonstration montre les capacités de rendu des formes géométriques du PDF Builder Pro.

Rectangle avec coins arrondis

Propriétés: Position (50,50), Dimensions (150x80), Fond rouge, Bordure noire 2px, Coins arrondis 10px
{ "type": "rectangle", "x": 50, "y": 50, "width": 150, "height": 80, "properties": { "fill": "#ff6b6b", "stroke": "#000000", "strokeWidth": "2px", "borderRadius": 10 } }

Cercle

Propriétés: Position (250,100), Dimensions (100x100), Fond turquoise, Bordure noire 2px, Rayon 50%
{ "type": "circle", "x": 250, "y": 100, "width": 100, "height": 100, "properties": { "fill": "#4ecdc4", "stroke": "#000000", "strokeWidth": "2px" } }

Ligne horizontale

Propriétés: Position (50,200), Longueur 200px, Épaisseur 3px, Couleur bleue
{ "type": "line", "x": 50, "y": 200, "width": 200, "height": 1, "properties": { "stroke": "#45b7d1", "strokeWidth": "3px" } }

Flèche directionnelle (droite)

Propriétés: Position (350,220), Dimensions (120x60), Fond jaune, Bordure noire 2px, Direction droite
{ "type": "arrow", "x": 350, "y": 220, "width": 120, "height": 60, "properties": { "direction": "right", "fill": "#f9ca24", "stroke": "#000000", "strokeWidth": 2 } }

Flèche directionnelle (haut)

Propriétés: Position (100,250), Dimensions (60x120), Fond violet, Bordure noire 2px, Direction haut
{ "type": "arrow", "x": 100, "y": 250, "width": 60, "height": 120, "properties": { "direction": "up", "fill": "#6c5ce7", "stroke": "#000000", "strokeWidth": 2 } }

Styles par défaut

Propriétés: Aucune propriété définie - utilisation des styles par défaut (fond transparent, bordure noire 1px)
{ "type": "rectangle", "x": 400, "y": 50, "width": 100, "height": 60 }

Types de formes supportés

  • rectangle: Rectangles avec support des coins arrondis
  • circle: Cercles et ellipses (utilise le minimum des dimensions)
  • line: Lignes droites horizontales (utilise border-top)
  • arrow: Flèches directionnelles (gauche, droite, haut, bas) en SVG

Propriétés communes:

  • fill: Couleur de fond (hexadécimal ou nom)
  • stroke: Couleur de bordure
  • strokeWidth: Épaisseur de bordure
  • opacity: Opacité (0.0 à 1.0)
  • strokeDasharray: Style de bordure (dashed, etc.)
  • borderRadius: Coins arrondis (rectangles uniquement)
  • direction: Direction de la flèche (right, left, up, down)

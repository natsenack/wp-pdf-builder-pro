# Fichiers temporaires de développement

Ce dossier contient tous les fichiers temporaires utilisés pendant le développement et les tests.

## Organisation

- `diagnostic/` - Scripts de diagnostic et monitoring
- `reports/` - Rapports de développement et analyses
- `scripts/` - Scripts de déploiement, FTP, et automatisation
- `tests/` - Fichiers de test temporaires et scripts de validation

Ces fichiers peuvent être supprimés en production ou archivés selon les besoins.

Test Flexbox - Footer contenu dans la modal

Cette version utilise flexbox avec overflow: hidden sur modal-content.

Structure :

  • modal-content: display: flex, flex-direction: column, overflow: hidden
  • modal-header: flex-shrink: 0 (reste en haut)
  • modal-body: flex: 1, overflow-y: auto (scroll si nécessaire)
  • modal-footer: flex-shrink: 0 (reste en bas)

Le footer devrait maintenant rester dans la modal même avec beaucoup de contenu.

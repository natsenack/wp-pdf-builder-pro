Professional high-performance PDF builder plugin for WordPress with advanced modular architecture.

== Description ==

PDF Builder Pro is a professional WordPress plugin for creating, customizing and generating PDF documents directly from your WordPress admin interface with powerful features and seamless integration.

Key Features:

* Visual drag-and-drop PDF template editor
* WooCommerce integration for automatic order PDF generation
* Customizable pre-defined templates
* Multilingual and localization support
* High-performance PDF generation engine
* Advanced asset management and intelligent caching
* Built-in backup and restore functionality
* Comprehensive analytics dashboard

== Installation ==

1. Download and unzip the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to "PDF Builder" in the main admin menu to configure your settings.

== Frequently Asked Questions ==

= Which WordPress versions are supported? =

PDF Builder Pro requires WordPress 5.0 or later.

= Is this compatible with WooCommerce? =

Yes, PDF Builder Pro provides native WooCommerce integration for automatic order PDF generation.

= What PDF formats are supported? =

The plugin generates standard PDF files compatible with all modern document readers.

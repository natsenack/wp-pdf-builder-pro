INSTRUCTIONS DE DIAGNOSTIC POUR L'ERREUR AJAX "APERÇU PDF"

Pour diagnostiquer l'erreur AJAX, veuillez :

1. Ouvrir la console du navigateur (F12)
2. Aller dans l'onglet Network
3. Cliquer sur le bouton "aperçu PDF"
4. Chercher la requête AJAX qui échoue
5. Copier l'URL complète et le message d'erreur

Vérifier aussi les logs PHP dans wp-content/debug.log

Les informations suivantes sont importantes :
- L'URL complète appelée
- Le code de statut HTTP (ex: 403 Forbidden, 500 Internal Server Error)
- Le message d'erreur exact
- Les logs PHP correspondants
